Please run the batch file "Copy-Files.bat" to copy the required DLLs and Doconut.lic (License file) to all the sample solutions.

======== OR =======

Open Sample solutions and do an "Add Reference" to the required DLLs from "Bin" Folder.

For ASP.NET 4.7+ / MVC, copy the Doconut.lic found in License folder to the respective solution's "Bin" folder.

For .NET Core place the Doconut.lic in the wwwroot directory.
